Eureka is a single column microblogging theme with customizable footer, menu and theme colors.

== Description ==

Eureka is intended for microblogging/journal sites that want to take advantage of WordPress Post Formats in a simple but customizable theme.
Eureka supports:

* All WordPress post formats. An icon corresponding to each format will be displayed with each post.
* Three widget areas in the footer.
* Customizable background and theme colors. 
* Multi-level menus.
* Translation.
* RTL language.

This theme is best suited for a screen resolution of 1024 x 768 and above though the layout will shrink to fit smaller screens.

Please visit the(WordPress repository page)[http://wordpress.org/extend/themes/eureka] for support/feature requests.

== Installation ==
'''Manual installation:'''
1. Download Eureka from the (Theme Repository)[http://wordpress.org/extend/themes/eureka]
2. Unzip the package
3. Upload the `eureka` folder to the `/wp-content/themes/` directory

'''Installation using "Add New Theme"'''
1. From your Admin UI (Dashboard), use the menu to select Appearance >> Install Themes
2. Search for 'eureka'
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button

'''Activation and Use'''
1. Activate the Theme through the 'Themes' menu in WordPress
2. Go to 'Menus' to set the header menu
3. Go to 'Widgets' to define content for the footer.
4. Customize background gradients by going to Appearance >> Customize
